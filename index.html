Space Invaders

Duration: 1 week, Individual

I chose this because it is one of the most popular games. I had to create a group of aliens who drop bombs in an interval and describe its movement. Similarly, I had to create a player with a laser which shoots as I press the shoot (space bar) button. An alien gets removed if it gets hit by the laser. Likewise, the player gets destroyed if hit by a bomb. The player has 3 lives and if all used the game is over. HTML displays scores, instructions and the game level. CSS is used to style the letters, aliens and the player. In JavaScript, I used the functions like alienMoving, dropBomb, startGame and endGame. The function uses setInterval where bombs are dropped in intervals. Similarly, removeAlien function removes aliens if hit by the player’s laser. I put a condition in the player's lives: if it is 0, the game ends, and if there are no more aliens, the player wins.

Languages and Technologies used:
HTML5 | CSS3 | JavaScript(ES6) | GitHub
